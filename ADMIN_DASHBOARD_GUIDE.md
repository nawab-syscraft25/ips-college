## Managing Content

### Pages Management

#### Creating a New Page
1. Click **Quick Actions ‚Üí Create Page** or go to **Pages** section
2. Enter page title and select the college
3. Set up SEO metadata (meta title, description, keywords)
4. Add social media sharing info (Open Graph)
5. Click **Create Page**

#### Editing a Page
1. Find the page in **Recent Pages** or **View All** in Pages section
2. Click **Edit** button
3. Modify title, slug, college, status
4. Update SEO metadata if needed
5. Click **Save Changes**

#### Page Design
1. From the dashboard or pages list, click **Design** button
2. Add sections (Hero, About, Stats, Courses, etc.)
3. Configure each section's content and styling
4. Preview changes in real-time
5. Save your changes

#### SEO Optimization
1. Click **SEO** from page actions
2. Fill in meta title (60 characters max)
3. Write meta description (160 characters max)
4. Add focus keywords
5. Configure Open Graph for social sharing
6. Set canonical URL if needed
7. Add schema markup for search engines

### Courses Management

#### Adding a Course
1. Click **Quick Actions ‚Üí Add Course**
2. Enter course details:
   - **Name**: Course title
   - **Department**: Academic department
   - **Level**: Diploma, UG, PG, etc.
   - **Duration**: Course length
   - **Fees**: Course cost
   - **Eligibility**: Admission requirements
3. Add detailed overview and curriculum
4. Set availability status (Active/Draft)
5. Submit

#### Managing Courses
- **View All Courses**: Click on the Courses stat card
- **Edit Course**: Select course and click Edit
- **Delete Course**: Mark as inactive or delete permanently
- **Search Courses**: Use the search feature to find specific courses

### Faculty Management

#### Adding Faculty Members
1. Click **Quick Actions ‚Üí Add Faculty**
2. Enter faculty details:
   - **Name**: Full name
   - **Designation**: Title/position
   - **Qualification**: Educational background
   - **Photo**: Upload profile image
   - **Bio**: Brief professional biography
3. Assign to college
4. Click Save

#### Managing Faculty
- View all faculty members
- Edit qualifications and bio
- Update profile photos
- Sort by college or department
- Search by name

### Facilities Management

#### Adding Facilities
1. Click **Quick Actions ‚Üí Add Facility**
2. Enter details:
   - **Name**: Facility name (Library, Lab, Gym, etc.)
   - **Description**: What it offers
   - **Image**: Upload facility photo
3. Assign to college
4. Publish

#### Viewing & Managing
- See all facilities in dashboard Overview
- Edit facility details anytime
- Update images for better visuals
- Manage multiple facilities per college

### Activities & Events

#### Creating Activities
1. Click **Quick Actions ‚Üí Add Activity**
2. Enter details:
   - **Title**: Event name
   - **Type**: Sports, Academic, Cultural, etc.
   - **Description**: Event details
   - **Image**: Event photo
   - **Date**: When it happened or will happen
3. Link to college
4. Publish

#### Organizing Events
- View upcoming and past activities
- Edit event details
- Update event photos
- Sort by type or date

### Applications & Enquiries

#### Viewing Applications
1. Go to **Recent Applications** on dashboard
2. See applicant name, course, and status
3. Click **View** to see full details

#### Managing Application Status
1. Open application details
2. Change status:
   - **Pending**: Under review
   - **Approved**: Selected
   - **Rejected**: Not selected
   - **Wait List**: Hold for future consideration
3. Save status

#### Handling Enquiries
1. Navigate to **Enquiries** section
2. View all inquiries from website visitors
3. Mark as handled/processed
4. Follow up with interested prospects

## Best Practices

### Content Management
‚úÖ **DO:**
- Use descriptive titles and slugs
- Fill in SEO metadata for all pages
- Upload optimized images (max 2MB)
- Keep content updated and relevant
- Organize content by college
- Test pages after publishing

‚ùå **DON'T:**
- Use special characters in slugs
- Upload huge image files

- Publish incomplete content
- Duplicate page titles
- Use outdated information

### SEO Optimization
1. **Meta Title**: 50-60 characters, include keywords
2. **Meta Description**: 150-160 characters, compelling
3. **Keywords**: 3-5 primary keywords per page
4. **URL Slug**: Short, descriptive, hyphenated
5. **Headings**: Use H1 once, H2/H3 for structure
6. **Images**: Include alt text and descriptive filenames

### Content Quality
- Proofread all content before publishing
- Use proper formatting and structure
- Add relevant images to enhance pages
- Keep information current
- Maintain consistent branding
- Use clear, concise language

## Troubleshooting

### Dashboard Not Loading
1. Clear browser cache
2. Refresh the page
3. Check database connection
4. Verify admin account exists
5. Check server logs

### Content Not Showing Up
1. Verify page/content is marked "Active"
2. Check college assignment
3. Ensure proper URL slug
4. Verify published date
5. Clear site cache if applicable

### Image Upload Issues
1. Check file size (max 5MB)
2. Verify file format (JPG, PNG, GIF)
3. Check file permissions
4. Try uploading to media library first
5. Use compressed images

### Permission Errors
1. Verify admin login
2. Check admin account status
3. Ensure proper database access
4. Verify file system permissions
5. Check server error logs
